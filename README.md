# Salesforce UI Clone

A responsive and visually appealing clone of the Salesforce landing page built using HTML and CSS. This project replicates the design, layout, and user interface of the official Salesforce website.

## 🚀 Features

* Fully responsive design
* Modern and clean user interface
* Navigation bar with dropdown-style layout
* Hero section with call-to-action buttons
* Embedded video section
* Multiple content sections
* Footer with organized links
* Optimized for desktop and mobile devices

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```bash
Salesforce-Landing-Page-Clone/
│── index.html
│── style.css
└── README.md
```

This project was created for learning and practicing frontend web development skills, including:
